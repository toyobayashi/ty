* Install

    ``` bash
    npm install
    ```

* Generate `.vscode/launch.json`

    ``` bash
    $ ty vscode
    ```

    Install `Debugger for Chrome` first.

* Start dev server

    ``` bash
    $ ty serve
    ```

* Launch app

    ``` bash
    $ ty start
    ```

* Start dev server and launch app. Auto restart if main process code changed.

    ``` bash
    $ ty dev
    ```

* Build code for production

    ``` bash
    $ ty build
    ```

* Pack app

    ``` bash
    $ ty pack
    ```

* Inspect webpack configuration

   ``` bash
   $ ty inspect
   ```
